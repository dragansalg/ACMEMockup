One might ask, what in the world is Scrum?


Scrum:

Development can be crazy and a way to make it less so is by using scrum.
This is a way to increase group productivity by making a clear schedule.
We plan how to work with eachother, divide the project into small tasks and decide when we will work with what.
We let each member pick up whichever task nobody else is working on.
When done with a task, make sure to check in with other members of the team to see if they are up to schedule and if they need any help.
